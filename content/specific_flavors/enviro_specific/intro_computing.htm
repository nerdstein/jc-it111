EXCEL Trend line Analysis….

MS Excel is a very nice tool for a wide range of data analysis.  Most, if not all of you have a reasonable amount of experience with Excel – perhaps through the IA course here at Juniata College.  Let’s consider an example of using Excel to predict monthly streamflow from monthly precipitation.  In this example, we are attempting create a simple model of streamflow from rainfall.  The data set :

Otter_river_flows.xls

is an Excel data set that contains a subset of rainfall and river flows for the Otter River, MI.  The ultimate goal is to be able to predict sediment flows into the lake at the lower end of the river.  Since sediment transport is very dependent on streamflow, the first step was to attempt to quantify streamflow based on rainfall – at least on a monthly basis.  To do this, we will develop a simple regression model or create a trend line in Excel. 

First let’s open the data set:

You will see April and May rainfall and runoff values (both are in cm) for the years 1963 to 1980.

You have plotted rainfall vs. runoff.  Now let’s add a trend line or a regression equation.  Go the CHART menu time and navigate to Add Trendline :

Let’s use a simple linear regression.  Then select the Options tab:

and opt to have the equation displayed and the R-squared.  Hit OK:

You have created a simple regression model!   The question is – “What does this mean?  How does this help me to predict flow?  What is X and what is Y?

In this case, the X is rainfall and the Y is flow and this linear regression model allows us to predict flow, Y, from Rainfall, X.  What is the R-squared value?
